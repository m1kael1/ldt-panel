# LDT Panel
## Requirements
- Node : v18.14.2
- NPM : v9.5.0 <br>


### First Step
```bash
git clone https://github.com/m1kael1/ldt-panel.git
```

```bash
cd ldt-panel
```

### Install Dependensi
```bash
npm install
```

### Untuk run development server
```bash
npm run dev
```
